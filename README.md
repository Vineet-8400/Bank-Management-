# 🏦 Bank Management System (C Project)

## 📌 Project Overview
This is a simple **Bank Management System** developed in C language using **Structures and File Handling**.  
It allows users to manage basic banking operations like account creation, deposit, withdrawal, and balance inquiry.

---

## 🚀 Features
- Create new bank account  
- Deposit money  
- Withdraw money  
- Check account balance  
- Store data permanently using file handling  

---

## 🛠️ Tech Stack
- Language: C  
- Concepts Used: Structures, File Handling, Pointers  
- Storage: Binary file (`bank.dat`)
